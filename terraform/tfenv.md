# tfenv


## Comandos

### intalar última versão terraform

#### exemplo

```shell
tfenv install latest
```


### intalar versão terraform

#### template

```shell
tfenv install <versao>
```

#### exemplo

```shell
tfenv install 1.3.9
```


### listar versoes instaladas

#### exemplo

```shell
tfenv list
```


### selecionar versão terraform

#### template

```shell
tfenv use <versao>
```

#### exemplo

```shell
tfenv use 1.3.9
```
